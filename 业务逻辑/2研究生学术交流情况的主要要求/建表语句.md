# 建表语句

## 实体表

### 学术交流活动

```sql
use StudentTest;

CREATE TABLE academic_communicate_activity
(
activity_name nvarchar(30)			NOT NULL,
student_number nvarchar(20) 		NOT NULL,
activity_location nvarchar(30) 		NOT NULL,
activity_date nvarchar(20) 			NOT NULL,
report_name_zh nvarchar(30) 		NOT NULL,
report_name_en nvarchar(50)			NOT NULL,
evidentiary_material nvarchar(50) 	NOT NULL,
audit_situation nvarchar(20) 		NOT NULL,
extra nvarchar(50) 					NOT NULL,
PRIMARY KEY (activity_name)
)
```
